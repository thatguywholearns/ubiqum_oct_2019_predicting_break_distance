Executive summary

The goal of this analysis is to predict the breaking distance of car based on its speed before breaking. We used a linear model to predict the breaking distance of the car. To get the best performing linear model we transformed the speed of the car. We squared the speed of the car, as the relationship between the squared speed and the breaking distance is quasi linear. We estimated the following model

\[y = 3.17 + 0.15x\]

In this equation x stands for the squared speed. This means that for every unit increase in squared speed of the car the breaking distance increase by 0.15. This means that the breaking distance increases at a quadratic rate for a constant speed increases. Or put in other words, the breaking distance becomes increasingly longer the faster you go.

The model perfromed well in terms of predicting test data, data never seen by the model before. The model explained 99% of the variance in the dependent variable. This can also be seen graphically as the graphical representation of the model closely fits the test datapoints.

Preprocessing

In order not to bias the prediction we remove this data point from the training data.

We create a scatter plot between the variables speed of car and distance of car in order to spot any type of relationship between the two variables.

We see that there is pattern present, however the pattern is not linear. Distance a car needs to break increases for each marginal speed increase, hinting towards a quadratic relationship. This also makes sense given that velocity and acceleration (or in this case decelartion, which is can be seen as negative acceleration) are described by a second degree polynomials.

In order to fit a line through the data that is able to predict the data we need to transform the distance variable. We square the variable so that the both variables increase at the same rate.

When we plot the the variable speed of car squared against the distance of car variable we see that relationship is linear.
